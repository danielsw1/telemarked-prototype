# Telemarked Prototype

Premium prototype for the Telemarked.no rebuild. Single self-contained `index.html` — no build step.

**Live:** https://danielsw1.github.io/telemarked-prototype/

Stack: Tailwind play CDN, GSAP + ScrollTrigger, Lenis smooth scroll, Inter font.

This is a design / interaction prototype, not a production site. All cart/checkout/account flows are visual only. Product data and testimonials are illustrative.
